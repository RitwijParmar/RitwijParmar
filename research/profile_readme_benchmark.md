# Profile README Benchmark

Profiles analyzed: **9**

## Patterns observed
- Strong profiles open with a one-line identity + current focus
- Sectioned layout with predictable scan path (Now, Projects, Writing/Contact)
- Evidence beats adjectives: use metrics, shipped links, demos
- Minimal but intentional visuals (one banner or project demos) outperform clutter
- Clear CTA and availability signal at bottom improves follow-through
- Automated freshness (recent activity/pinned sync) keeps profile credible
